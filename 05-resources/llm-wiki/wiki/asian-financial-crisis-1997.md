# 亚洲金融危机 (1997-1998)

## 概况

1994 年底到 1998 年，短短 4 年间，墨西哥、东南亚、韩国、俄罗斯、巴西等数十个国家接连爆发金融危机。亚洲持续十多年的经济奇迹在一年之内崩盘。视频以上帝视角梳理了整条危机传播链。

- **来源**：小Lin说，B站 BV1hN596zEas
- **核心主线**：资本的跨境流动 —— 涌入 → 泡沫 → 撤离 → 崩盘

## 危机传播链条

### 第一站：墨西哥 (1994)

- **背景**：美国低利率 (3%) → 美元资本涌入墨西哥（汇率绑定，5% 收益，无汇率风险）
- **泡沫**：每年约 300 亿美元流入，GDP 飞涨
- **触发**：美联储加息至 5% → 墨西哥投资优势消失 → 资本撤离
- **挣扎**：墨西哥发美元债 + 10% 高利率 → 借新还旧 → 外汇储备耗尽
- **结局**：94 年 12 月比索与美元脱钩，3 天暴跌 40%
- **后遗症**：IMF 500 亿美元救助 → 资本产生"IMF 会兜底"的道德风险

### 第二站：东南亚 (1996-1997)

- **背景**：日本利率极低 + 日元升至历史高位 → 日元套息交易涌入东南亚
- **规模**：日本银行占泰国债务 50%、印尼 39%、马来西亚 36%、韩国 25%
- **热钱两个问题**：
  - 短视：只投金融和房地产，吹起泡沫
  - 跑得快：日本国内银行危机（坏账 2300 亿美元）→ 被迫撤离 → 东南亚现金流被抽干
- **新增变数**：空头资本趁火打劫

### 索罗斯的做空策略

三步走：
1. **提前布局**：买看跌期权、做空期货
2. **扛旗进攻**：用海量资金正面做空外汇/期货市场
3. **聚集合力**：市场发现"这国家真快不行了"→ 跟风做空 → 恐慌崩盘 → 获利离场

历史案例：1992 年做空英镑（黑色星期三），狂赚 18 亿美元。

关键条件：必须找到"真的快不行了"的国家，大家才会跟进。固定汇率国家是最优目标——做空失败汇率不变不亏，成功则暴跌暴赚。

### 泰国战役 (1997)

- 96 年：小资本试探进攻泰铢，失败撤退
- 97 年：索罗斯和老虎基金进场 → 纽约伦敦法兰克福投行跟风
- 高盛 5 月发报告"泰铢面临重大贬值压力"→ 吹响进攻号角
- 泰国反击：隔夜拆借利率拉至 1500%（做空两星期亏一半）
- 6 月大决战：索罗斯砸 10 亿，老虎基金 30 亿 → 泰国最大金融公司破产
- **7 月 2 日**：泰铢与美元脱钩，几天内暴跌 50%，亚洲金融危机全面爆发

### 香港金融保卫战 (1997-1998)

- **第一次进攻** (97.8)：游资试探，港股跌约 10%，游资撤退
- **第二次进攻** (97.10)：更猛烈攻势，隔夜拆借利率暴涨至 300%，港币守住但港股跌 40%
- **第三次进攻** (98.8)：空头总结规律——做空港币 → 利率涨 → 股市跌，双向埋伏做空仓位
- **破天荒反击**：香港金管局忍无可忍，直接入市买入港股
- **8 月 28 日**：单日成交量 790 亿港元（平时 20 倍），买家几乎全是金管局
- **结果**：两周用掉超 1000 亿港元，空头撤退，港股守住

### 俄罗斯 (1998)

- 特点：政府外债以卢布计价（GKO），不是美元债
- 亚洲危机 → 油价暴跌 → 卢布国债被疯狂抛售 → GKO 利率飙至 150%
- 利息支出超过全国税收 → 陷入严重的债务危机
- IMF 数十亿美元贷款被腐败吞噬
- **8 月 17 日**：俄罗斯宣布卢布贬值、国债违约
- 连锁反应：LTCM 暴雷 → 美联储救市 → 全球新兴市场债券被抛售 → 巴西崩盘 → 油价跌至不到 10 美元 → 中东遭殃

## 底层逻辑：资本流动的"一进一飘一撤"

```
热钱涌入 → 吹起金融和房地产泡沫 → 
某种触发（加息/银行危机）→ 热钱撤离 → 
空头进攻 → 恐慌加剧 → 汇率/股市崩盘 → 
信贷危机 → 输入性通胀 → 长期衰退
```

## 关键洞察

1. **资本不是单一主体**：养老金、银行、对冲基金各怀目的，利益互相冲突
2. **索罗斯体量远小于被攻击国家**：他靠的是"找到快不行的国家 + 扛旗聚集合力"
3. **IMF 兜底的道德风险**：救墨西哥让资本觉得"高收益低风险"，变本加厉涌入
4. **越自由越需要监管**：90 年代的教训让华盛顿共识彻底消失
5. **新加坡、台湾、香港挺住了**：自身制度健全的国家受到攻击能较快恢复，说明危机的根在内部

## 与交易框架的关联

索罗斯的做空方法论（找到快不行的国家 → 扛旗进攻 → 聚集合力）与 [[multi-agent-trading-framework|多智能体交易决策框架]] 中的风险识别逻辑一致：交易的关键是识别"已确认的脆弱性"，而非预测未来。亚洲金融危机中空头的成功，本质上是发现了"固定汇率 + 热钱依赖"这一结构性弱点——这相当于交易框架中"已确认事实 > 预期"原则的宏观版演绎。
