## 行业调研笔记（2026-05-22）

> 调研目的：在回复甲方前，先建立对中国电视剧海外版权出口业务的系统认知。

### 一、中剧出口三大模式及占比

按影视制作方采用的渠道统计（云合数据《2023/2024中国剧集报告》）：

| 模式 | 占比 | 说明 |
|------|------|------|
| **直签海外平台/电视台** | ~40%+ | 版权方直接谈 Netflix、Disney+、Amazon、各国电视台 |
| **平台自营国际版** | ~30% | 腾讯WeTV、爱奇艺国际版、优酷国际版、芒果TV国际版 |
| **其他**（代理中介、YouTube分账等） | ~30% | 代理买断某地区版权再分销 |

**注脚**：
- 以上按"采用该渠道的制作方数量"统计，非出口金额或剧集数量。同一公司可同时走多条渠道。
- 按出口剧集总数量看，平台自营可能占比最高——优爱腾芒的自制/定制剧出海时优先保留到自己国际版，盘子最大。
- **代理模式正在萎缩**。从业者透露："现在所有长视频平台、大大小小的影视公司都要自己做海外发行，不让我们中间商赚钱了。"片方甚至会将最赚钱的泰国市场自留，只把次要市场给代理。

### 二、非S级剧不能同步播出的原因

排序不分先后，**翻译本地化是最大瓶颈**：

| 原因              | 具体表现                                                        | 成本谁承担           |
| --------------- | ----------------------------------------------------------- | --------------- |
| **翻译+配音+物料本地化** | 古装剧翻译折损大；长剧人工译制每分钟300-500元，高质量版1900-3300元/分钟；海报、预告片需按国际标准重做 | 买断→买方出；自营→国内平台出 |
| **供片速度**        | 海外平台对物料有严格技术标准，国内后期完成后需重新输出适配版本                             | 片方              |
| **版权谈判时机**      | 很多剧的海外版权在国内播出前不久才签约，来不及同步准备宣发                               | 各付各的            |
| **平台排播策略**      | Netflix、Disney+ 各有排播节奏，不会为中国剧专门调档期（S级除外）                    | 机会成本            |
| **演员海外宣传档期**    | 同步播出通常需主演配合海外宣传                                             | 片方/平台协调         |

**审查不是主因**——国内审查在国内播出前已完成，拿到发行许可证的剧在审查层面不存在出海障碍。

**核心结论**：延迟不是因为某一方"不愿意出钱"，而是非S级剧的海外预期收益撑不起同步播出所需的翻译+物料+宣发投入。

### 三、出口参考价格

#### 3.1 价格是按照什么范围报的？

**单一地区买家价 ≠ 全球打包价**。

- 2026年3月香港影视展数据："单一买家最高出价单集$10万，全部买家合计达单集$20万"——即泰国（或某个单一高购买力市场）一个买家出$10万/集，所有地区加起来才能到$15-20万/集。
- Netflix、Disney+ 一般是全球/多地区打包买，单价反而不如分地区卖的总和高。Netflix出价"并不高"，Disney+相对高一些。

#### 3.2 各层级出口价格（每集，美元）

| 层级 | 单价 | 案例 |
|------|------|------|
| **顶配S级** | $10-15万（单一买家）<br>$15-20万（多买家合计） | 某85花古装剧$10万/集×40集=$400万（新丽副总裁王乔公开演讲） |
| **A+级流量剧** | $3.5-10万 | 肖战《骄阳伴我》>$3.5万/集 |
| **A级剧** | $1-3万 | 出海主力古偶/甜宠 |
| **普通剧** | $5千-1万 | 泰国市场中等级别 |
| **尾部剧** | <$5千 | 星光弱的剧整部可能只卖十几万人民币 |

#### 3.3 按市场细分

| 市场 | 单价($/集) | 备注 |
|------|-----------|------|
| 泰国 | $0.5-1万（普通），$1万+（好剧） | 购买力最强，中剧市占45% |
| 日本 | 有时高于泰国 | 挑剧极严 |
| 台湾 | $0.1-0.2万 | 2018年曾达$1万+/集，已大幅下滑 |
| Disney+ | 开价高于Netflix | 深度参与宣推 |
| Netflix | 开价不高 | 品牌价值 > 商业收入 |
| 越南 | 越来越低 | 盗版+政策限制 |

#### 3.4 北美单独卖，价格会大幅下调吗？

**会，幅度不小。** 原因：

1. **东南亚是中剧出海核心收入来源**，泰国占中剧海外流媒体份额45%。之前的$10万/集等数字，驱动力主要来自东南亚。北美单独拆出来，买家少、竞争弱。
2. **美国传统电视台几乎不买中剧**，只走流媒体。买方寡头（Netflix、Disney+、iQiyi/WeTV北美版），价格被压。
3. **北美非华语受众对中剧没有持续消费习惯**，和韩剧差距巨大。买方采购预算中分配给中剧的比例很小，中剧是在跟韩剧抢同一块预算。
4. **公开数据找不到北美单独定价**——这本身就是信号，它不是独立的重要市场。

**粗略估算**：A级剧在泰国可卖$1-3万/集，北美单独卖可能只有$几千/集。很多剧的版权方根本不单独拆北美，跟着全球打包走更省事。

### 四、对本次咨询的关键启示

1. 진명이형 问的 "미국판매 가격"，如果仅限美国地区版权，真实价格远低于行业报道里的$10万/集等数字，需要帮他校准预期。
2. 我的网大/短剧背景反而是优势——短剧出海（ReelShort、DramaBox）在美国比长剧活跃得多，2024年海外短剧市场规模突破500亿人民币。如果 진명이형 对短剧也有兴趣，切入口更大。
3. 追问清单里的地区范围（仅美国 vs 北美 vs 全美洲）和版权类型（独家 vs 非独家）对价格影响极大，必须等他回复后才能给出有意义的价格区间。

## 甲方视角：引进中剧的操作路径

> 假设 진명이형 的角色是：把中国电视剧版权卖给 KOCOWA 平台的代理商（买方代理）。

### 他的定位

不是中国版权方的代理，而是 **KOCOWA 的买方代理**——用自己对 KOCOWA 需求的理解和对中国市场的触达能力，帮 KOCOWA 找到合适的中剧、谈好价格、完成交易。

### 完整链条

#### 第一步：搞清楚中国剧的版权在谁手里

一部中国新剧的海外版权归属，取决于它的出身：

| 剧集来源 | 美国版权通常谁说了算 | 找谁 |
|----------|-------------------|------|
| 爱奇艺/腾讯/优酷/芒果自制剧 | 平台自己持有全球版权 | 平台国际版权部门 |
| 独立制作公司出品（非平台定制） | 制作公司持有，可能已卖给某代理 | 制作公司 + 确认是否已售出 |
| 已卖给华策/酷看等代理的剧 | 代理商持有 | 华策 C-dramaRights、酷看文化等 |
| 平台参投但非独家的剧 | 制作公司和平台共有，分地区谈判 | 两边都可能要谈 |

**现实难点**：没有公开的"中剧版权挂牌清单"。得通过人脉一层层问，才知道某部剧的美国版权还在不在、谁说了算。

#### 第二步：接入信息流

需要知道"现在有什么新剧、谁在做、美国版权还在不在"。渠道：

- **行业展会**：香港 FILMART（3月）、戛纳 MIPTV/MIPCOM、新加坡 ATF、莫斯科 World Content Market——这些是中国版权方集中卖剧的地方
- **平台关系**：直接联系爱优腾的国际版权部门
- **中方中间人**：帮你打听哪些剧还没卖出美国

#### 第三步：撮合交易

1. 向中方版权方询价（确认美国地区还在、报价）
2. 拿报价去和 KOCOWA 确认是否有兴趣、预算是否匹配
3. 如果 KOCOWA 有兴趣，推进正式谈判（独家/非独家、窗口期、权益范围、付款条件）
4. 签约 → 提供物料（翻译字幕由谁负责要谈）→ KOCOWA 上线

#### 第四步：他的独特价值（为什么中方要理他？）

不是传话筒，必须有至少一个不可替代的价值点：

| 价值点 | 可行性 |
|--------|--------|
| **KOCOWA 的确定采购预算** — "我带着买家来的，不是来询价的" | ⭐ 最强。告诉中方"我代表KOCOWA，有真实预算，今天就能出offer"，比平台自营海发团队在展会上扫街高效得多 |
| **韩语字幕 + 韩式运营** — KOCOWA 用户以韩裔美国人为主，中剧进 KOCOWA 需要韩语字幕而非英语，这在市场上是稀缺能力 | ⭐ 差异化优势 |
| **韩剧+中剧打包卖** — 如果同时掌握韩剧供应，可以跟 KOCOWA 谈"韩剧搭中剧"的打包协议，中方看到的是一个有渠道控制力的买家 | ⭐ 现有核心优势 |
| **韩国市场的溢出** — 如果一部中剧适合进韩国（不仅美国），可同时代理韩国发行，对中方一个交易覆盖两个市场 | 加分项 |
