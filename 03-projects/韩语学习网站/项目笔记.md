# 韩语学习网站

一站式韩语学习工具，包含词汇（Vocabulary）、语法（Grammar）、跟读（Shadowing）三大模块。

## 设计文档

- [[05-resources/llm-wiki/outputs/韩语学习网站-2026-05-21]] — 完整开发记录：词汇体系 → 语法板块 → 复习功能 → Shadowing 搭建 → 重构排错 → UI 国际化

## 访问方式

- 本地文件：`file://` 打开 `index.html`（Vocabulary + Grammar 可用）
- 本地服务器：`python3 server.py` → `http://localhost:8766`（含 Shadowing）

## 关键文件

- `index.html` — 主应用
- `server.py` — 本地服务器（/api/download 端点，Shadowing 依赖）
- `regenerate_html.py` — 从 JSON 数据源重新生成 HTML
- `batch1-7.json` — 词汇数据
- `grammar_study4.json` / `grammar_quiz4.json` — 语法数据
